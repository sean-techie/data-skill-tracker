# Workforce Skills Analytics Dashboard

A data analytics dashboard built with Python, Pandas, Streamlit, and Plotly to analyze workforce skill demand across industries.

## Features

- Skill demand analysis
- Interactive visualizations
- Industry filtering
- Workforce insights
- Data exploration dashboard

## Tech Stack

- Python
- Pandas
- Streamlit
- Plotly
